# CLAUDE.md

## Project Overview

Personal learning repository by Dr. Chirag Khant for exploring GitHub and version control concepts.

## Guidelines

- Keep files simple and minimal — this is a learning repo, not a production project
- Do not over-engineer solutions
- Prefer editing existing files over creating new ones
- Never auto-commit — always ask before committing or pushing

## Communication Style

- Keep responses short and clear
- Explain concepts simply when teaching GitHub/git topics
- Avoid unnecessary jargon

## Workflow

- Always confirm before making destructive git operations (reset, force push, branch delete)
- Use descriptive commit messages
